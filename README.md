# 🐱 猫猫世界大分身

一个有趣的猫咪MBTI测试Web应用。

## 项目结构

- `index.html` - 主页面
- `js/` - JavaScript文件
  - `catMap.js` - 猫咪映射
  - `quizBank.js` - 问题库
  - `stamp.js` - 印章相关
- `css/` - 样式文件
  - `styles.css` - 主样式
- `images/` - 图片资源
  - `cat_png/` - 猫咪图片
- `assets/` - 其他资源
  - `music1.mp3` - 背景音乐

## 使用

直接在浏览器中打开 `index.html` 即可运行。

## 功能

- MBTI测试
- 生成个性化猫咪卡片
- 分享功能
